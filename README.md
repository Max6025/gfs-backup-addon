# GFS Backup – Home Assistant Addon

Großvater-Vater-Sohn Backup-Strategie für Home Assistant.

**Täglich, wöchentlich, monatlich und jährlich** – mit automatischer Rotation  
**lokal auf HA und auf dem NAS** (Synology, QNAP, FritzNAS, Windows Share usw.)

## Features

- ✅ 4 Backup-Ebenen: täglich / wöchentlich / monatlich / jährlich
- ✅ Lesbare Dateinamen: `HA-Daily-2026-06-27.tar`
- ✅ Getrennte NAS-Unterordner pro Ebene
- ✅ Automatische Rotation lokal **und** auf NAS
- ✅ Optionale Backup-Verschlüsselung
- ✅ Alles in einer Addon-Konfiguration einstellbar
- ✅ Kein Blueprint, kein externes Skript nötig

## Installation

1. Repository URL in HA hinzufügen:
```
https://github.com/DEIN-GITHUB/gfs-backup-addon
```

2. Addon installieren, konfigurieren, starten
